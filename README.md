# Asr.ai v2
Full frontend + lightweight Express backend.
## Quick start
1. Copy `.env.example` to `.env` and set `OPENAI_API_KEY`.
2. Install:
   ```bash
   npm install
   ```
3. Run:
   ```bash
   npm run dev
   ```
4. Visit http://localhost:3000
## Deploy to Vercel
- Push to GitHub and import in Vercel. Set `OPENAI_API_KEY` in project settings.
